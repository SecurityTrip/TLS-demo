# Демонстрация реального TLS 1.3 на Python

Простая, но полностью настоящая реализация TLS 1.3 с помощью встроенного модуля ssl.

## Что показывает демо

- Настоящий TLS 1.3 handshake
- Только TLS 1.3 (принудительно, без fallback на 1.2)
- ECDHE — эфемерный обмен ключами (Perfect Forward Secrecy)
- Шифрование AES-256-GCM или ChaCha20
- Аутентификация сервера с помощью сертификата
- Проверка сертификата клиентом
- Полностью зашифрованный обмен сообщениями

Всё работает на localhost, без интернета.

## Файлы проекта
```
.
├── generate_cert.py    → создаёт самоподписанный сертификат (запустить один раз)
├── server.crt          → сертификат сервера
├── server.key          → приватный ключ сервера
├── server.py           → TLS 1.3 сервер с красивыми логами
├── client.py           → TLS 1.3 клиент с красивыми логами
└── README.md           → этот файл
```

## Как запустить

1. Убедитесь, что у вас Python 3.8+
   python --version

2. Установите зависимость
   `pip install cryptography`

3. Сгенерируйте сертификат (один раз)
   `python generate_cert.py`

4. Запустите сервер (в первом терминале)
   `python server.py`

5. Запустите клиент (во втором терминале)
   `python client.py`
